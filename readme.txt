The commands for react will be listed below:
# Note: After initializing the react app and navigating to it's directory
    # Install react-router-dom
        npm install react-router-dom
    # Start app
        npm start

The commands for python are:
# Note: After navigating to directory
    # Setup venv
        python -m venv venv
    # Run script
        myenv\Scripts\active
    # Install Flask
        pip install Flask
    # Install Flask_CORS
        pip install flask-cors
    # Finally run the run app
        python app.py
